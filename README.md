Currency Converter Website
Overview-
This is a Currency Converter website created as a college project. The website allows users to explore a user-friendly interface for converting currencies.
The project focuses on webpage design, layout, and user interface skills.

Features-
Clean and intuitive interface.
Multiple currency options displayed.
Responsive design (works on desktop and mobile devices).
Styled buttons, inputs, and dropdowns for an interactive look.

Technologies Used-
HTML5 – For webpage structure.
CSS – For styling, layout, and responsive design.

How to Use-
Open the index.html file in a web browser.
Explore the design and layout of the currency converter interface.
The interface demonstrates visual structure and styling for currency conversion.

Project Structure-
currency-converter/
│
├── index.html   # Main webpage
├── style.css    # Styling for the webpage
└── README.md    # Project description

Learning Outcomes-
Practice in HTML and CSS design.
Understanding of layout, styling, and responsive design.
Experience in building structured web pages.
